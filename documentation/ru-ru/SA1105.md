## Причина

Конструкции в выражении запроса C# занимает несколько строк и начинается не со своей строки.

## Описание правила

Нарушение этого правила происходит, когда конструкция запроса охватывает несколько строк, но не начинается со своей строки. Например:
```c#
    var elementDescriptions =
        from element in elements select GenerateElementDescription
        (
            element
        );
```

Правильно конструкции запроса можно записать в следующем виде:
```c#
    var elementDescriptions =
        from element in elements
		select GenerateElementDescription
        (
            element
        );
```
 
## Как исправить нарушения

Чтобы исправить нарушение этого правила, переместите конструкцию вниз, чтобы она начиналась со следующей строки.

## Как подавить нарушения

```c#
#pragma warning disable SA1105 // Конструкции запроса, занимающие несколько строк, должны начинаться со своей строки
    var elementDescriptions =
        from element in elements select GenerateElementDescription
        (
            element
        );
#pragma warning restore SA1105 // Конструкции запроса, занимающие несколько строк, должны начинаться со своей строки
```
