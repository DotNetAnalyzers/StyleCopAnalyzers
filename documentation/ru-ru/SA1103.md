## Причина

Конструкции в выражении запроса C# не размещаются на одной строке, и каждая конструкция не размещается на своей строке.

## Описание правила

Нарушение этого правила происходит в том случае, если конструкции запроса не размещены на одной строке или каждая из них не размещена в своей строке. Например:
```c#
    object x = from num in numbers
        select num;
```

Правильно конструкции запроса можно записать как:
```c#
    object x = from num in numbers select num;
```
или:
```c#
    object x =
        from num in numbers
        select num;
```
 
## Как исправить нарушения

Чтобы исправить нарушение этого правила, убедитесь, что все конструкции расположены вместе на одной строке, или каждый конструкция начинается в отдельной строке.

## Как подавить нарушения

```c#
#pragma warning disable SA1103 // Конструкции запроса должны быть в отдельных строках или все в одной строке
    object x = from num in numbers
        select num;
#pragma warning restore SA1103 // Конструкции запроса должны быть в отдельных строках или все в одной строке
```
