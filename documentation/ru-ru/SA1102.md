## Причина

Конструкция запроса C# не начинается с той же строки, что и предыдущая, или со следующей строки.

## Описание правила

Нарушение этого правила происходит, когда конструкция в выражении запроса начинается не с той же строки, что и предыдущая конструкция, или не со строки после конструкции запроса. Например:
```c#
    object x = from num in numbers

        select num;
```

Предложение запроса может быть правильно записано как:
```c#
    object x = from num in numbers select num;
```
или:
```c#
    object x =
        from num
        in numbers
        select num;
```
 
## Как исправить нарушения

Чтобы исправить нарушение этого правила, убедитесь, что каждая конструкция в выражении запроса начинается с той же строки, что и предыдущая конструкция, или со следующей строки.

## Как подавить нарушения

```c#
#pragma warning disable SA1102 // Конструкция запроса должна следовать за предыдущей конструкцией
    object x = from num in numbers

        select num;
#pragma warning restore SA1102 // Конструкция запроса должна следовать за предыдущей конструкцией
```
