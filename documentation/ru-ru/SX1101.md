## SX1101

<table>
<tr>
  <td>TypeName</td>
  <td>SX1101DoNotPrefixLocalMembersWithThis</td>.
</tr>
<tr>
  <td>CheckId</td>
  <td>SX1101</td>
</tr>
<tr>
  <td>Категория</td>
  <td>Правила удобочитаемости</td>
</tr>
</table>

## Cause

Локальный вызов члена экземпляра класса или базового класса имеет префикс `this.`.

## Описание правила

Нарушение этого правила происходит всякий раз, когда код содержит локальный вызов члена экземпляра класса или базового класса, который имеет префикс `this.`. Исключение составляет случай, когда имя параметра метода конфликтует с идентификатором после префикса `this.`, в этом случае о нарушении не сообщается.
По умолчанию это правило отключено. При включении этого правила должно быть отключено правило [SA1101](SA1101.md).

## Как исправить нарушения

Чтобы исправить нарушение этого правила, удалите префикс 'this.' перед обращением к члену класса.

## Как подавить нарушения

```csharp
#pragma warning disable SX1101 // DoNotPrefixLocalMembersWithThis
#pragma warning restore SX1101 // DoNotPrefixLocalMembersWithThis
```
