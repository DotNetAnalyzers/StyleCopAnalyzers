## Причина

Конструкция в выражении запроса C# начинается с той же строки, что и предыдущая конструкция, когда предыдущая конструкция занимает несколько строк.

## Описание правила

Нарушение этого правила происходит, когда конструкция запроса охватывает несколько строк, а следующее конструкция начинается на той же строке, что и конец предыдущей конструкции. Например:
```c#
    var elementNames =
        from element in GetElements
        (
            12,
			45
        ) select element.Name;
```

Правильно конструкции запроса можно записать в следующем виде:
```c#
    var elementNames =
        from element in GetElements
        (
            12,
			45
        )
		select element.Name;
```
 
## Как исправить нарушения

Чтобы исправить нарушение этого правила, переместите конструкцию вниз, чтобы она начиналась со следующей строки.

## Как подавить нарушения

```c#
#pragma warning disable SA1104 // Конструкция запроса должна начинаться с новой строки, если предыдущая конструкция охватывает несколько строк
    var elementNames =
        from element in GetElements
        (
            12,
			45
        ) select element.Name;
#pragma warning restore SA1104 // Конструкция запроса должна начинаться с новой строки, если предыдущая конструкция охватывает несколько строк
```
