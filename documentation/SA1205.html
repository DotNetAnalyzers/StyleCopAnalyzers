TypeName

PartialElementsMustDeclareAccess

CheckId

SA1205

Category

Ordering Rules

Cause

The partial element does not have an access modifier defined. StyleCop may not be able to determine the correct placement of the elements in the file. Please declare an access modifier for all partial elements.

Rule Description

A violation of this rule occurs when the partial elements does not have an access modifier defined.

How to Fix Violations

To fix an instance of this violation, specify an acess modifier for the partial element.

How to Suppress Violations

[SuppressMessage("StyleCop.CSharp.OrderingRules", "SA1205:PartialElementsMustDeclareAccess", Justification = "Reviewed.")]
